simple-transfar-search-extension
================================

ジョルダンの乗換検索をChromeの検索バーから簡単に使えるようにする拡張機能

## インストール方法

現在Chrome Web Storeの方には登録していません。

- リポジトリをcloneする
- chrome://extensions より「デベロッパーモード」をONにする
- パッケージ化されていない拡張機能を読み込む
- cloneしたディレクトリを選択

この手順により手元のブラウザにインストールが可能です。
